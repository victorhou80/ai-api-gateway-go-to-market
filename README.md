# Kimi + GPT-5.5 AI API Gateway Go-To-Market Playbook

This repository is a practical go-to-market package for a compliant AI API gateway service that currently supports Kimi and GPT-5.5.

The positioning is deliberately narrow:

> Use Kimi for low-cost Chinese-language bulk work, use GPT-5.5 for high-value reasoning and final output quality, and use the gateway to control cost, keys, routing, usage, budgets, invoices, and customer delivery.

This is not an API recharge, key resale, or region-bypass business.

## What This Package Covers

- ICP definition and customer prioritization.
- Concrete acquisition paths for early customers.
- Outreach scripts for AI tools, implementation studios, developers, and ecommerce/content teams.
- The first sellable offer: a Kimi + GPT-5.5 cost-routing assessment.
- Pricing and packaging for pilot deployment and monthly managed service.
- Singapore invoice and China invoicing considerations.
- Operating checklist for the first 30 days.

## Business Boundaries

Do:

- Sell API gateway SaaS, technical implementation, routing, billing, and managed service.
- Support customer-owned keys where possible.
- Use prepaid service contracts.
- Provide usage reports, routing analysis, budget controls, and access governance.
- Work with customers that have legitimate business use cases and clear billing requirements.

Do not:

- Sell or transfer third-party API keys.
- Sell API credits or balances as standalone goods.
- Share accounts, sessions, credentials, or verification codes.
- Promise access that bypasses model-provider region, payment, KYC, or usage restrictions.
- Compete with grey-market "cheap API" resellers.

## Recommended First Offer

Start with a low-friction assessment instead of selling the full gateway immediately:

> Kimi + GPT-5.5 Cost Routing Assessment

Customer provides 10-30 sanitized request samples and rough monthly usage. You classify tasks into Kimi-suitable and GPT-5.5-required categories, estimate cost savings, then propose a 7-day pilot deployment.

Suggested pricing:

- Free or RMB 999 for first assessment.
- RMB 2,999 for deep assessment.
- RMB 5,000-15,000 for a 7-day pilot deployment.
- RMB 2,000-8,000/month for managed gateway operations.

## 30-Day Target

The first 30 days should focus on market validation, not platform expansion:

- 300 qualified leads collected.
- 180 direct outreaches sent.
- 25 effective replies.
- 8 diagnostic calls.
- 5 sanitized sample sets received.
- 3 assessment reports delivered.
- 1-2 paid pilot deployments.

If these numbers do not produce conversations, revise ICP and messaging before adding more models or building more product features.
