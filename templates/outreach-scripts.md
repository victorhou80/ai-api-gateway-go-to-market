## AI Tool Site

```text
你好，我看你们产品已经有 AI 生成/问答能力。

1. 普通中文任务走 Kimi，降低 API 成本；
2. 关键输出走 GPT-5.5，保证质量；
3. 按用户/项目统计调用量，设置预算、限流和异常告警。

如果你们现在每月 API 成本超过 3000 元，我可以免费帮你做一次模型路由评估：
看哪些请求适合走 Kimi，哪些必须保留 GPT-5.5，大概能省多少钱。

可以先不用接入，给我 10 条脱敏后的典型请求样本就行。
```

Follow-up after 2 days:

```text
补充一下，我们不是让你们换掉现有模型。

更像是在你们现有调用前面加一层路由：

摘要/分类/初稿 -> Kimi
复杂推理/重要输出/终稿 -> GPT-5.5

这样可以先从成本最高的接口开始优化，不影响主业务。
```
