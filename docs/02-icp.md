# Ideal Customer Profile

ICP means Ideal Customer Profile: the customer type most likely to feel pain, understand the offer, pay quickly, and produce a useful case study.

## One-Sentence ICP

The first-stage ICP is:

> A small AI product team, content/ecommerce team, or AI implementation studio that already uses model APIs, spends RMB 3,000-50,000/month on AI calls, mainly handles Chinese text, and needs routing, budgets, usage reports, key control, and cost reduction without buying grey-market keys or API credits.

## Priority ICP 1: AI Tool Sites And Small SaaS

| Attribute | Detail |
|---|---|
| Examples | AI writing tools, customer support bots, knowledge-base Q&A, resume tools, paper polishing, product copy tools |
| Team size | 1-20 people |
| Current state | AI feature is live, likely already paying for model API calls |
| Monthly model cost | RMB 3,000-50,000 |
| Decision maker | Founder, independent developer, technical lead |
| Pain | Rising model cost, abuse, key exposure, unclear user-level usage, no budget control |
| Offer | Kimi + GPT-5.5 routing, usage reports, per-user budgets, retry/fallback, managed gateway |
| Entry | Free cost-routing assessment |

This is the strongest first ICP because they already understand APIs and can adopt a gateway quickly.

## Priority ICP 2: AI Implementation Studios

| Attribute | Detail |
|---|---|
| Examples | Dify/FastGPT/RAG implementers, AI app developers, software outsourcing firms |
| Team size | 3-50 people |
| Current state | Repeatedly build AI apps for clients |
| Pain | Rebuilding model access, key management, limit controls, logs, and billing for each project |
| Offer | White-label gateway base layer |
| Entry | One project pilot |

This ICP can become a channel because one studio may introduce multiple client projects.

## Priority ICP 3: Content And Ecommerce Teams

| Attribute | Detail |
|---|---|
| Examples | Ecommerce operators, cross-border sellers, content teams, private-domain marketing teams |
| Team size | 3-50 people |
| Current state | Employees already use AI tools, but usage is fragmented |
| Pain | Unclear spend, inconsistent quality, no team-level control |
| Offer | Kimi for bulk drafts and GPT-5.5 for final high-value copy |
| Entry | Sample workflow run with real content |

Do not sell this group "API gateway" first. Sell workflow, quality, and cost control.

## Secondary ICP: Training And Community Operators

This can work if framed as a course lab or teaching sandbox.

Avoid letting the offer become "cheap API key distribution."

| Good Framing | Bad Framing |
|---|---|
| Course API lab | Cheap model key |
| Student quota control | Resold API credits |
| Teaching workflow | Account sharing |
| Usage records | Unlimited access |

## Non-ICP

Do not prioritize these leads:

- Individuals who only ask for the cheapest API price.
- Anonymous buyers.
- Customers asking for account sharing, proxy access, or region bypass.
- Large enterprises with heavy procurement before you have references.
- Pure model hobbyists asking for many providers that you do not yet support.
- Grey-market reseller operators.

## Lead Scoring

Use a 10-point score.

| Condition | Score |
|---|---:|
| Already has AI functionality | +2 |
| Has paid product or commercial usage | +2 |
| Text, customer support, knowledge-base, copywriting, or content workflow | +2 |
| Clear contact method | +1 |
| Small team rather than large platform | +1 |
| Estimated model API cost above RMB 3,000/month | +2 |
| Only asks for cheap API credits | -5 |
| Requests bypass, account sharing, or key resale | Reject |

Prioritize leads scoring 7 or higher.
